# SCIN 모델 재학습 가이드 - 개선판 (Balanced Focal Loss)

## 📋 재학습 배경

### 기존 모델 성능
- **손실 함수**: BCEWithLogitsLoss + 클래스 가중치
- **F1-Score**: 0.160
- **Top-5 Accuracy**: 62.1%
- **문제점**: Threshold 0.5가 너무 높아 예측 보수적

### Focal Loss 재학습 실패 (2024-12-04)
- **손실 함수**: Focal Loss (alpha=0.25, gamma=2.0)
- **F1-Score**: 0.085 ❌ (47% 성능 저하)
- **문제점**: Major class (Eczema/ACD) 과적합
  - Eczema Recall: 100%, Precision: 32% (모든 샘플을 Eczema로 예측)
  - 파라미터가 너무 강해서 역효과 발생
- **결과**: 아카이브 처리 (`archive/focal_loss_failed/`)

### 개선된 재학습 전략
- **손실 함수**: Balanced Focal Loss (alpha=0.5, gamma=1.5)
- **변경사항**:
  - `alpha`: 0.25 → **0.5** (minor class 보호 강화)
  - `gamma`: 2.0 → **1.5** (major class 과적합 완화)
  - Learning Rate: 0.0001 → **0.00005** (더 안정적)
  - Batch Size: 16 → **32** (일반화 향상)
  - Dropout: 0.6 → **0.3** (과적합 완화)
- **목표**: F1-Score **0.20~0.30** (기존 대비 25~87% 향상)

## 🔧 상세 설정

### 주요 하이퍼파라미터

| 파라미터 | 기존 (실패) | 개선판 | 설명 |
|---------|-----------|-------|------|
| **손실 함수** | Focal Loss | Balanced Focal Loss | 파라미터 완화 |
| **alpha** | 0.25 | **0.5** ↑ | Minor class 가중치 증가 |
| **gamma** | 2.0 | **1.5** ↓ | 쉬운 샘플 무시 완화 |
| **Learning Rate** | 0.0001 | **0.00005** ↓ | 더 안정적 학습 |
| **Batch Size** | 16 | **32** ↑ | 일반화 향상 |
| **Dropout** | 0.6 | **0.5** ↓ | 과적합 완화 |
| **Weight Decay** | 1e-3 | **1e-4** ↓ | L2 정규화 완화 |
| **Epochs** | 50 | **60** | 충분한 수렴 시간 |
| **Patience** | 15 | **10** | Early stopping |

### Focal Loss 수식
```
FL(p_t) = -α(1-p_t)^γ log(p_t)

where:
  p_t = 예측 확률
  α = 클래스 가중치 (0.5 권장)
  γ = focusing parameter (1.5 권장)
```

**작동 원리:**
- `(1-p_t)^γ`: 쉬운 샘플(p_t 높음)의 loss를 줄임
- `α`: Minor class에 더 많은 가중치 부여
- **γ가 낮을수록** major class 과적합 방지
