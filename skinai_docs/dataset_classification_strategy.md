## 3. 모델 구조 — 공유 백본 + 분리 헤드

### 단일 모델 vs 분리 모델 비교

| | 단일 21-class 모델 | 완전 분리 모델 2개 | **공유 백본 + 분리 헤드** |
|---|---|---|---|
| 정확도 | 낮음 (도메인 혼재) | 높음 | 높음 |
| 배포 복잡도 | 단순 | 복잡 (모델 2개 관리) | 중간 |
| 학습 데이터 효율 | 낮음 | 낮음 | **높음** |
| 임상 라우팅 | 없음 | 별도 라우터 필요 | **내장됨** |
| 도메인 갭 대응 | 미흡 | 완벽 | 충분 |

**결론: 공유 백본 + 분리 헤드 구조를 권장**

### 아키텍처

```
입력 이미지
    │
    ▼
┌─────────────────────────────┐
│   Shared Backbone           │  ← EfficientNet-B3 / ConvNeXt
│   (피부 저수준 특징 학습)      │     두 데이터셋으로 공동 학습
└─────────────────────────────┘
    │
    ▼
┌─────────────────────────────┐
│   Domain Router Head        │  ← 3-class
│   염증성 / 종양성 / 정상      │
└─────────────────────────────┘
    │                   │
    ▼                   ▼
┌──────────┐     ┌──────────────────────┐
│ DS14 Head│     │ DS15 Head            │
│ 5-class  │     │ 15-class             │
│ (염증성)  │     │ (종양성 + 위험도)     │
└──────────┘     └──────────────────────┘
```

### 공유 백본을 선택하는 이유

1. **도메인 갭 대응**: DS14(안면) ↔ DS15(전신)의 촬영 환경 차이를 헤드 분리로 흡수
2. **데이터 부족 완화**: DS15 악성 클래스는 샘플이 적은데, DS14 데이터로도 피부 표현 학습 가능
3. **파라미터 효율**: 완전 분리 모델 대비 백본 파라미터를 공유해 모델 사이즈 절감
4. **임상 트리아지 내장**: Domain Router가 1차 분류 역할 수행

## 4. 학습 전략

```
Phase 1 — 백본 사전학습 (Backbone Pretraining)
  - ImageNet pretrained 백본 로드
  - DS14 + DS15 전체 데이터로 Domain Router 학습 (3-class)
  - 백본은 freeze 상태 유지

Phase 2 — 헤드별 학습 (Head Training)
  - DS14 Head: DS14 데이터만으로 5-class 학습
  - DS15 Head: DS15 데이터만으로 15-class 학습
  - Focal Loss 적용 (악성 클래스 샘플 수 불균형 대응)

Phase 3 — 전체 Fine-tuning (End-to-end)
  - 백본 unfreeze + 낮은 learning rate (백본: 1e-5, 헤드: 1e-4)
  - DS14:DS15 샘플 비율 균형 조정 (WeightedRandomSampler)
  - 악성 클래스에 loss weight 추가 적용
```

## 5. 주의할 혼동 쌍

### 데이터셋 간 혼동 (이름 유사)

| DS14 클래스 | DS15 클래스 | 주의 사항 |
|-------------|-------------|-----------|
| 지루**피부염** | 지루**각화증** | 이름 유사하나 완전히 다른 병변. class name 명확히 구분 필요 |
| 여드름 (안면, 붉은 구진) | 화농 육아종 (혈관성 구진) | 시각적으로 유사할 수 있음 |

### Dataset 15 내부 혼동 (색소성 병변)

| 쌍 | 위험도 차이 |
|----|-------------|
| 악성흑색종 ↔ 멜라닌세포모반 | 악성 vs 양성 → 가장 중요한 혼동 |
| 악성흑색종 ↔ 흑색점 | 악성 vs 추적 필요 |
| 기저세포암 ↔ 지루각화증 | 악성 vs 양성 |
| 광선각화증 ↔ 지루각화증 | 전암성 vs 양성 |
| 혈관종 ↔ 화농 육아종 | 양성 vs 양성 (혈관성) |

> 모델 평가 시 이 쌍들의 혼동 행렬(Confusion Matrix)을 중점 모니터링

## 6. Dataset 15 — 계층적 분류 (DS15 단독 모델 필요 시)

DS15만 별도로 다룰 경우 2단계 계층 분류를 사용한다.

```
Stage 1: Risk Triage (3-class)
├── 악성 (Malignant)      → 악성흑색종, 기저세포암, 편평세포암
├── 전암성 (Precancerous) → 보웬병, 광선각화증
└── 양성 (Benign)         → 나머지 10개

Stage 2: 세부 분류
├── 악성 3-class  → 악성흑색종 / 기저세포암 / 편평세포암
├── 전암성 2-class → 보웬병 / 광선각화증
└── 양성 10-class → 세부 분류
```
